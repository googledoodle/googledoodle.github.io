Matplotlibとは

matplotlibは、Pythonでグラフを描画したり、イメージを表示させたりする際に使用するライブラリです。matplotlibを使うと、様々な種類のグラフを描画することができます。主に2次元のグラフですが、3次元のグラフも描画することができます。
